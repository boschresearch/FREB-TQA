# Dataset structure
The three sub-folders correspond to the three robustness aspects we proposed:

#### 01_str ➡️ Retrieval robustness in case of table structure changes

- four files correspond to four datasets (WTQ, Wikisql, SQA and TAT), containing only extraction questions.

#### 02_rel ➡️ Attention to relevant cells

- two files correspond to two datasets (WTQ and TAT), containing only reasoning questions.


#### 03_agg_com ➡️ Aggregation/comparison robustness in case of value changes

- eight files correspond to two datasets (WTQ and TAT).
  
- each dataset includes four files:
    * answer change
  
    * answer not change
  
    * short table
  
    * original table
